streamhub-deck

Overview

StreamHub Deck, A "TweetDeck"-esque columned layout of content from StreamHub.

Prerequisites

  • Bower - A package manager for JavaScript libraries

Running the demo

  1. git clone git@github.com:cheung31/streamhub-deck.git
  2. cd streamhub-deck
  3. bower install
  4. python -m SimpleHTTPServer 8888 (or another server of your choice)
  5. Navigate to localhost:8888 (or the appropriate port number)

More about Livefyre StreamHub

For more information about Livefyre StreamHub, visit http://livefyre.com/streamhub.
